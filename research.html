Existing Research

A curated collection of seminal and recent papers that underpin this project. Organized by research theme to help you build a coherent understanding of the field.

Diffusion Language Models

Foundational and recent work on discrete diffusion models for text generation.

2024 NeurIPS

Simplified and Generalized Masked Diffusion for Discrete Data (MDLM)

Sahoo, S., Arriola, M., Schiff, Y., Gokaslan, A., Marroquin, E., Chiu, J.T., Rush, A., Kuleshov, V.

Introduces MDLM, a masked discrete diffusion model demonstrating strong performance on language generation benchmarks. Key reference for the modeling approach used in this project.

Discrete Diffusion Language Modeling Core Reference
View on arXiv
2021 NeurIPS

Structured Denoising Diffusion Models in Discrete State-Spaces (D3PM)

Austin, J., Johnson, D.D., Ho, J., Tarlow, D., van den Berg, R.

Introduces D3PM, a family of diffusion models for discrete data including text, laying the groundwork for modern discrete diffusion approaches. Essential for understanding MDLM's foundations.

Discrete Diffusion Foundational
View on arXiv
2024 ICML

Masked Diffusion Language Models are Fast and Scalable Learners

Ye, T., Zheng, H., Chen, W., Ma, Z., Wang, O., Gu, Q.

Presents an efficient and scalable masked diffusion architecture. Demonstrates that diffusion LMs can compete with autoregressive models on perplexity while being much faster at inference.

Masked Diffusion Scalability Efficiency
View on arXiv
2023 ICLR

DiffuSeq: Sequence to Sequence Text Generation with Diffusion Models

Gong, S., Li, M., Feng, J., Wu, Z., Kong, L.

Applies continuous diffusion models to conditional text generation tasks like machine translation and summarization. Relevant as a comparison point for the sequence-to-name generation framing.

Seq2Seq Conditional Generation
View on arXiv

Code Intelligence

Pre-trained models and LLMs specifically designed for source code understanding and generation.

2020 EMNLP

CodeBERT: A Pre-Trained Model for Programming and Natural Languages

Feng, Z., Guo, D., Tang, D., et al.

Introduces CodeBERT, a bimodal transformer pre-trained on code and natural language pairs. A foundational baseline for code-related NLP tasks and understanding of identifier semantics.

Pre-trained Model Code Understanding
View on arXiv
2021 EMNLP

CodeT5: Identifier-aware Unified Pre-trained Encoder-Decoder Models for Code

Wang, Y., Wang, W., Joty, S., Hoi, S.C.H.

CodeT5 uses identifier-aware pre-training objectives, making it directly relevant to the identifier renaming task. Used as a key baseline in this project's evaluation.

Encoder-Decoder Identifier-aware Key Baseline
View on arXiv
2023 arXiv

StarCoder: May the Source Be With You!

Li, R., Allal, L.B., Zi, Y., et al.

StarCoder is a large FIM-capable code LLM trained on The Stack dataset. Relevant as a direct FIM-baseline for code completion and identifier prediction tasks.

FIM Large LLM Baseline
View on arXiv
2024 arXiv

DeepSeek-Coder: When the Large Language Model Meets Programming

Guo, D., Zhu, Q., Yang, D., et al.

DeepSeek-Coder is a state-of-the-art code-focused LLM supporting FIM. Used as a strong baseline in this project for fill-in-the-middle identifier generation.

FIM SOTA Baseline
View on arXiv

Identifier Naming in SE

Research specifically on naming quality, identifier renaming, and related refactoring.

2024 ICSE

RefineID: Refining Code Identifier Names in the Wild

[Authors TBD – internal reference]

Directly benchmarks identifier renaming using real-world developer commits. Provides the primary dataset and evaluation framework used in this project. Essential reading.

Identifier Renaming Benchmark Core Reference
Used in this project
2022 TSE

A Survey on Code Naming Smells: Definition and Detection

Peruma, A., Mkaouer, M.W., et al.

Systematically defines naming smells in software (linguistic patterns that signal poor naming) and surveys automatic detection approaches. Motivates the need for the automatic renaming task.

Naming Smells Survey
View Paper
2023 FSE

BAGGER: Leveraging Variable Name Context for Code Variable Name Suggestions

Liu, S., et al.

Presents a context-aware approach to suggesting better variable names using surrounding code tokens. Directly related to the contextual masking strategy in our project implementation.

Variable Naming Context-aware
View Paper
2021 ASE

Learning to Suggest Better Variable Names from Code Diffs

Alon, U., Sadaka, R., Levy, O., Yahav, E.

Uses a graph neural network approach to learn from code diffs how developers rename variables. A classic baseline for the identifier renaming task using graph-structured code representations.

GNN Code Diffs Renaming
View on arXiv

Evaluation Methods

Metrics and frameworks for evaluating code generation and LLM performance.

2023 NeurIPS

Judging LLM-as-a-Judge with MT-Bench and Chatbot Arena

Zheng, L., Chiang, W.L., Sheng, Y., et al.

Proposes the LLM-as-judge evaluation paradigm, using strong LLMs to assess model outputs. We adopt this approach as a supplementary metric alongside EM for renaming quality evaluation.

LLM Judge Evaluation MT-Bench
View on arXiv
2002 ACL

BLEU: a Method for Automatic Evaluation of Machine Translation

Papineni, K., Roukos, S., Ward, T., Zhu, W.

The classic BLEU metric for evaluating generated text quality against references. Used alongside EM to provide a more nuanced evaluation of identifier renaming predictions.

BLEU Metric Classic
View Paper
2023 arXiv

OctoPack: Instruction Tuning Code Large Language Models

Muennighoff, N., Liu, Q., Zebaze, A., et al.

Introduces HumanEvalPack, a multilingual extension of HumanEval for evaluating code generation across languages. Useful context for understanding code LLM evaluation methodology.

Benchmarking Code Eval
View on arXiv
