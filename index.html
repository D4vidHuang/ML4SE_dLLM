ml4se_dLLM.py 
from diffucoder import DiffuCoder
import torch

# dLLM-based code refactoring
class AutoRefactorer:
    def __init__(self):
        self.model = DiffuCoder
        self.task = "code_refactoring"

    def refactor(self, code_ctx):
        # Mask & diffuse code tokens
        masked = self.mask_tokens(code_ctx)
        return self.model.generate(masked)

About the Course

Why ML4SE?

Software engineering is increasingly empowered by AI. This course bridges the gap between modern machine learning, particularly large language models and diffusion-based approaches, and real-world software engineering tasks.

Students work on cutting-edge research problems, gaining hands-on experience with state-of-the-art methods and contributing original insights to the field.
